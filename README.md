## 【功能介绍】
**分为三种角色，管理员，教师，学生。
管理员:设置教师，学生，院系，班级，课程排课等。
教师:查看课程信息，学生名单及选课情况，录入成绩与查询。
学生:查看课程信息，选课操作，退课操作，查看选课记录，课程成绩等信息。**
